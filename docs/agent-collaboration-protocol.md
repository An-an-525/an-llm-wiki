# 安的书房 Agent 协作协议

这份文档用于本地协作，不是公网展示内容。

## 参与者

| 名称 | 角色 | 主要产物 |
| --- | --- | --- |
| 小鲸鱼（DeepSeek TUI） | 本地资料采集与事实提取 | 事实卡、证据摘要、风险标记 |
| 小安 | 内容筛选、中文改写、读者分层 | 公开候选稿、资料包、问答边界 |
| Codex | 前端、后端、构建、验收、发布 | 代码改动、数据同步、检查报告 |
| 审阅子代理 | 挑刺和复核 | 问题清单、修正建议 |

## 三层边界

| 层 | 用途 | 规则 |
| --- | --- | --- |
| raw | 保留原始证据 | 只读优先，不直接发布 |
| compiled-private | 脱敏摘要和候选内容 | 可给小安和 Codex 使用 |
| public | 前端和 App 展示 | 只放中文、可读、可复刻、安全内容 |

## 交接格式

```yaml
任务: 简短标题
上下文: 为什么要做
边界: 可读范围、可写范围、不得公开的信息
已读材料:
  - 材料摘要
建议动作:
  - 下一步动作
需要谁接手: 小鲸鱼 | 小安 | Codex | 审阅子代理
验收方式:
  - 如何判断完成
```

## 协作流程

1. 任务先进任务板。
2. 小鲸鱼只读采集，输出事实和风险。
3. 小安把事实改写成中文资料包候选。
4. Codex 把候选内容接入前端或数据层。
5. 审阅子代理挑刺。
6. Codex 跑构建、内容质量、隐私和浏览器验收。
7. 结果写回任务板。

## 安全例外

- secret-risk、local-path-risk、raw-only 任务不得进入前端数据。
- 密钥、token、SSH、cookie、session、账号状态只允许记录类型、位置、哈希和处理建议。
- 公开页面只讲任务、方法、结论、边界和验收，不公开完整调用轨迹。
