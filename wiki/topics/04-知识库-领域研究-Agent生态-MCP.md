# 🔌 MCP (Model Context Protocol)

> MCP is an open standard that enables AI models to interact with local and remote data sources and tools.

---

## 📊 核心数据

- **GitHub**: [modelcontextprotocol/servers](https://github.com/modelcontextprotocol/servers)
- **Stars**: 83.3k ★
- **Forks**: 10.3k
- **类别**: **基础设施 / 协议**
- **收费情况**: **完全开源 (Apache-2.0)**

---

## 🛠️ 功能特性

- **标准化接口**: 解决了 AI 助手与不同软件/硬件之间重复开发连接器的问题。
- **生态化发展**: 已经拥有数百个社区维护的 Server（如 Google Drive, Slack, GitHub, SQLite 等）。
- **客户端支持**: Claude Desktop, Cursor, Zed 等主流 AI 编辑器均已原生支持。

## 🚦 社区评价

- **优势**: 由 Anthropic 发起并捐赠给 Linux 基金会，已成为行业事实标准，极大降低了 Agent 扩展工具的成本。
- **劣势**: 需要一定的编程基础来配置或开发 Server。
