## 🛠️ 功能特性

- **多层记忆**: 支持用户、会话、Agent 多维度的记忆存储。
- **自动内化**: 自动从对话中提取事实并更新记忆。
- **跨平台一致性**: 确保 Agent 在不同平台访问相同的用户记忆。

## 🚦 社区评价

- **优势**: 专注于“记忆”这一垂直领域，解决了 Agent 长效交互的痛点。
- **劣势**: 接入需要额外的存储开销，且记忆清洗逻辑需调优。
