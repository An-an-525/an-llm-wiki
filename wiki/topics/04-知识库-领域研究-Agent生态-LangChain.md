# 🦜 LangChain

> LangChain is a framework for developing applications powered by large language models (LLMs).

---

---

## 🛠️ 功能特性

- **可组合性**: 通过 Chain 机制连接 LLM 与其他组件。
- **丰富的集成**: 支持几乎所有主流 LLM、向量数据库和工具。
- **生态系统**: 包含 LangSmith (监控) 和 LangServe (部署)。

## 🚦 社区评价

- **优势**: 行业基石，社区极其活跃，资源和教程最丰富。
- **劣势**: 架构由于过度封装显得臃肿，学习曲线较陡，部分组件由于迭代过快存在不稳定性。
