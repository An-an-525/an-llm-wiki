# 🦙 LlamaIndex

> LlamaIndex (formerly GPT Index) is a data framework for your LLM applications. It provides the tools to build agentic applications over your data.

## 🛠️ 功能特性

- **数据连接器 (Data Connectors)**: 支持从 PDF、API、数据库等多种源摄取数据。
- **数据索引 (Data Indexes)**: 将非结构化数据转化为可查询的知识结构。
- **查询接口 (Query Interface)**: 提供高级的 RAG 查询能力，支持对话式交互。
- **Agentic RAG**: 支持具备自主决策能力的 RAG 系统，能够根据查询需求动态选择检索策略。

## 🚦 社区评价

- **优势**: RAG 领域的绝对领导者，对复杂文档处理和索引能力极强。
- **劣势**: 随着功能不断扩展，API 变得越来越复杂，对于简单应用可能显得过于重型。
