# AI Agent 系统总览

## 系统拓扑

```
用户（安）
  ├── 手机小安 📱 — KimiCLaw（总调度）
  └── 电脑
       ├── 笔记本小安 🖥️ — KimiCLaw Desktop
       ├── opencode 💻 — 工头/执行引擎（deepseek-v4-pro）
       ├── Claude Code 🔧 — 架构师（claude-4）
       └── Codex 🌐 — 浏览器自动化（离线）
            │
       Obsidian Wiki（共享大脑）[local path redacted]
            │
       ~/.multi-agent/（邮箱 + 注册表 + 策略）
```

## 技术栈

| 层 | 技术 |
|----|------|
| Agent 引擎 | KimiCLaw(k2.6) / Claude Code / opencode(deepseek) |
| 知识中枢 | Obsidian + 11k 笔记 + 3 仪表盘 |
| 通信 | CLI 互调(kimi/claude -p) + 文件邮箱 + Obsidian 信号 |
| 浏览器 | Firecrawl(搜/抓/交互) + Playwright(自动化) |
| 视觉 | Kimi k2.6(image_in + video_in) |
| 自动化 | HEARTBEAT.md(心跳) + agent-wrapper.sh(重试) |
| 版本控制 | Git + gh CLI |

## 通信协议

```
实时: kimi --quiet -p / claude -p (同步 CLI 互调)
异步: ~/.multi-agent/mailbox/{agent}/inbox/ (文件邮箱)
信号: [local path redacted] (Obsidian)
状态: ~/.multi-agent/runtime/registry.json (Agent 注册表)
约束: ~/.multi-agent/runtime/policy.json (机器可执行规则)
锁: ~/.multi-agent/runtime/locks/ (乐观锁 + 30min 超时)
```

## 自动化任务

1. KimiCLaw 心跳(30min) — Agent 存活 + 失败日志 + Wiki 巡检
2. 会话自动沉淀 — 每次任务完成 → 写 Wiki
3. 视觉自动路由 — 图片 → KimiCLaw 处理
4. 失败重试 — 3 次上限 → 升级 KimiCLaw → 通知用户
