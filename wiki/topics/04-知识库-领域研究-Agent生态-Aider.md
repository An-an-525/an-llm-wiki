# ⌨️ Aider

> AI Pair Programming in Your Terminal. Aider allows you to edit code in your local git repository using LLMs.

---

- **类别**: **编程 Agent / 结对编程**
- **收费情况**: **开源 (Apache-2.0)**

---

## 🛠️ 功能特性

- **Git 原生**: 自动管理 Git 提交，每一次变更都会生成有意义的 Commit 信息。
- **全代码库感知**: 通过 Repository Map 机制，让 LLM 能够理解整个项目的结构。
- **极简安装**: 仅需 `pip install` 即可开始，支持 100+ 种编程语言。
- **自举开发**: Aider 70% 的代码是由 Aider 自身编写的。

## 🚦 社区评价

- **优势**: 速度极快，与 Git 工作流结合完美，是目前最实用的终端编程工具。
- **劣势**: 仅限命令行界面；代码直接在主机执行（无沙箱隔离）；在处理超大规模跨文件重构时，相比 Claude Code 稍显逊色。
