# 👐 OpenHands

## 🛠️ 功能特性

- **企业级特性**: 提供 Web UI、VSCode 集成、RBAC (基于角色的访问控制) 及审计日志。
- **沙盒运行**: 基于 Docker 的代码执行沙箱，确保主机安全性。
- **多 Agent 协作**: 支持复杂任务的多 Agent 委派与执行。
- **商业背书**: 获得了大量融资，并被 AMD、Apple、Google 等企业采用。

## 🚦 社区评价

- **优势**: 生态完整，支持 Web 和 IDE 两种模式，安全性高。
- **劣势**: 资源开销大（至少 4GB RAM，本地运行需要 16GB VRAM）；在处理完全未知的问题上仍有提升空间。
