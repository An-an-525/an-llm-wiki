# 🤖 AutoGen

> AutoGen is a framework for creating multi-agent AI applications that can act autonomously or work alongside humans.

## 🛠️ 功能特性

- **多 Agent 会话**: 支持多种 Agent 交互模式（1对1, 群聊, 嵌套）。
- **代码执行**: 具备内置的安全代码执行沙箱。
- **人机协作**: 易于集成人类反馈到 Agent 工作流中。

## 🚦 社区评价

- **优势**: 微软背书，研究深度高，在多 Agent 协作逻辑上非常灵活。
- **劣势**: 生产环境稳定性曾受诟病（现已有改善），配置相对复杂。
