# ⚙️ n8n 技能

> Manage n8n workflows and automations via API. Use when working with n8n workflows, executions, or automation tasks.

---

## 🛠️ 核心功能

- **工作流管理**: 列出、获取详情、创建、激活/禁用工作流。
- **执行监控**: 查看执行历史、获取特定执行的详细状态。
- **性能优化**: 分析瓶颈、获取优化建议、生成健康报告。
- **自动化测试**: 支持验证工作流结构及 Dry Run 测试。

---

## 📖 使用方法

### 命令行示例
```bash
# 列出活跃工作流
python3 scripts/n8n_api.py list-workflows --active true --pretty

# 验证工作流结构
python3 scripts/n8n_tester.py validate --id <workflow-id>

# 分析性能瓶颈
python3 scripts/n8n_optimizer.py analyze --id <workflow-id>
```

---

## ⚠️ 关键原则 (Workflow Creation Rules)

- **完整性**: 始终生成包含所有功能节点（HTTP Request, Code, Set 等）的完整工作流。
- **禁止占位符**: 严禁创建仅包含 TODO 注释或设置说明的非功能性节点。
- **真实连接**: 确保所有节点之间都有正确的逻辑连接。
