# 代码片段索引

> `03 - 代码片段` 是代码复用的 canonical 区，不再把片段分散到其他平级目录。

## 推荐分类

- Shell / PowerShell
- Python
- TypeScript / Node
- SQL
- Regex / 检查表达式

## 命名建议

- `<技术栈>-<用途>.md`
- 例：`powershell-obsidian-audit.md`
- 例：`python-frontmatter-normalize.md`

## 使用规则

- 只收“可复用的最小片段”，不要把整份项目文档搬进来
- 每个片段最好说明：用途、输入、输出、风险
- 与具体项目强绑定的代码，仍优先放回项目目录或项目总览的附属文档
