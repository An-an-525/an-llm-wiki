## 📚 技能库概述

技能库是用户构建的 AI 技能集合，用于扩展 AI Agent 的能力。每个技能都是针对特定任务的专用工具。

### 技能分类

| 分类 | 说明 | 示例 |
|------|------|------|
| **开发工具** | 编程和开发相关 | GitHub, Docker, 代码分析 |
| **知识管理** | 笔记和知识相关 | Obsidian, 文档处理 |
| **数据处理** | 数据分析和处理 | Excel, CSV, JSON |
| **系统管理** | 系统操作和维护 | 文件管理, 系统监控 |

---

## 🔧 核心技能列表 (07 - 技能库)

### 1. [[07 - 技能库/GitHub|GitHub 技能]]
- **用途**: 与 GitHub 仓库交互。
- **核心功能**: 仓库管理、Issue/PR 追踪、代码差异分析。
- **调用方式**: `@skill://github [action] [params]`

### 2. [[07 - 技能库/Obsidian|Obsidian 技能]]
- **用途**: 自动化操作 Obsidian 知识库。
- **核心功能**: 原子化读写、全库检索、双向链接维护。
- **调用方式**: `@skill://obsidian [action] [params]`

### 3. [[07 - 技能库/文档处理|文档处理技能]]
- **用途**: 跨格式办公文档解析与转换。
- **核心功能**: PDF/Word 文本提取、Excel 表格识别、内容智能摘要。
- **调用方式**: `@skill://document-processor [action] [params]`

### 4. [[07 - 技能库/MuleRun_Skills_Hub|MuleRun 技能库 (扩展)]]
- **用途**: 提供 85 个专业 Agent 技能的完整知识参考。
- **核心功能**: 涵盖 Browser-Use, Playwright, Data-Pipeline, API-Integration 等多维度自动化能力。
- **调用方式**: 参考内部具体的技能定义片段。

---

## 🎯 技能使用指南

### 调用协议
使用 `@skill://[技能标识符]` 语法调用。Agent 在调用前必须阅读对应技能的详细定义文件。

### 组合工作流示例
1. 使用 `GitHub 技能` 获取最新代码提交。
2. 使用 `文档处理技能` 生成变更说明。
3. 使用 `Obsidian 技能` 将说明沉淀至项目日志。

---

## 📝 技能开发规范

### 1. 结构化定义
新技能必须在 `07 - 技能库/` 下创建独立的 Markdown 文件，且包含完整的 YAML Frontmatter。

### 2. 核心要素
- **功能概述**: 明确技能的边界与能力。
- **参数说明**: 详细列出必填与可选参数及示例值。
- **使用限制**: 标注速率限制、权限要求及安全性警告。
