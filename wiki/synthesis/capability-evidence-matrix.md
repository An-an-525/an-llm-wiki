# Capability Evidence Matrix

This matrix keeps capability information concrete. A capability should explain what can be delivered, what evidence supports it, and what workflow can be repeated.

## Claim Standard

Every public capability claim should have:

- a public page or source note;
- a project where the capability was exercised;
- a repeatable procedure;
- a known failure mode.
