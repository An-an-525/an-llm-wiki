# llm-wikid

This repository is a secondary implementation reference for Obsidian-oriented vault conventions. It does not replace Karpathy's `llm-wiki.md` gist as the official pattern source.

## Use In This Vault

- Borrow only practical vault-template ideas that fit the public-safe structure.
- Keep `_raw/`, private inbox material, and volatile Obsidian state outside Git.
- Validate every publishable page through the local structure and privacy scripts.
